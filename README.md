# UI To Code

## Overview

A simple UI-to-Code tool that reads text from a screenshot and generates a basic HTML interface.

---

## Features

- Screenshot text extraction
- OCR support
- Basic UI component detection
- HTML generation

---

## Installation

```bash
pip install pillow pytesseract
```

---

## Usage

```bash
python code_generator.py
```

---

## Example

Input Screenshot:
